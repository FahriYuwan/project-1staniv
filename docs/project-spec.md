# Project Overview

This project is a romantic anniversary storytelling website.

The purpose is to create an emotional digital experience celebrating
the 1 year anniversary between Fahri Yuwan and Anggi.

The website should feel like an interactive memory book.

Users navigate through memories chapter by chapter.

The experience must feel warm, nostalgic, and personal.

---

# Tech Stack

Framework: Next.js (App Router)
Language: TypeScript
Styling: Tailwind CSS
Animation: Framer Motion

The website must be mobile-first.

Primary target device is mobile.

Desktop layout should adapt gracefully.

---

# Core Features

The website must include:

1. Password gate before entering the website
2. Background music autoplay
3. Chapter based storytelling
4. Navigation arrows
5. Hamburger menu for chapter navigation
6. Countdown timer
7. Photo galleries
8. Love letter section
9. Hidden final emotional page
