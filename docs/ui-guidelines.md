# UI Principles

The website must feel like a digital memory book.

Avoid generic landing page layouts.

Use soft animations and warm tones.

---

# Layout Rules

Each chapter fills the entire viewport height.

Content is centered vertically.

Photos should have rounded corners and soft shadows.

---

# Animation

Use Framer Motion.

Allowed animations:

fade in
slide transition
soft hover scale
popup scale animation

Avoid heavy animations.

---

# Mobile UX

Mobile is the primary design target.

Ensure:

touch friendly buttons
no text overflow
readable typography

---

# Performance

Use next/image for photos.

Lazy load images.

Avoid large libraries.
